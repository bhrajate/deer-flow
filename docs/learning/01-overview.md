# 外围认知

## 项目定位

DeerFlow 2.0 是一个开源的 **super agent harness（超级智能体运行底座）**，基于 LangGraph 与 LangChain，为智能体提供完成实际工作的通用基础设施，而不只是提供问答能力。

- **解决的问题**：开发者构建能执行复杂、多步骤任务的智能体时，需要自行拼装工具、文件系统、执行环境、记忆、上下文管理和多智能体协作；DeerFlow 把这些能力集成成可直接使用且可扩展的运行底座。
- **主要用户**：希望直接使用超级智能体的用户，以及需要定制技能、工具、模型和执行环境的智能体应用开发者。
- **产品演变**：1.x 主要面向 Deep Research；2.0 是不共享旧代码的重写，从研究框架扩展为通用智能体底座。
- **典型任务**：深度研究、数据流水线、报告或幻灯片生成、网页与仪表盘制作、内容工作流自动化。

### README 展示的核心能力

1. **Skills & Tools**：用 `SKILL.md` 封装工作流和领域知识，按需渐进加载；内置工具之外还可通过 MCP 和 Python 扩展。
2. **Sub-Agents**：主智能体可拆解任务，生成上下文和工具相互隔离的子智能体，并行工作后汇总结果。
3. **Sandbox & File System**：每个任务拥有文件系统和执行环境，可操作上传文件、工作区和最终产物，而非只生成文本。
4. **Context Engineering**：通过上下文隔离、摘要和文件卸载支撑长时间、多步骤任务。
5. **Long-Term Memory**：跨会话保存用户偏好和积累的知识，并由用户本地控制。

### 当前理解

这里的 **harness** 可以理解为“智能体运行底座”：它不规定智能体只能完成某一种业务，而是负责组织模型、工具、技能、状态、执行环境和子智能体，使上层任务能够真正落地。
